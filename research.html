Software Security

Human Factor

Human Factor is the process of making human and computer interaction as easy as possible.

Usability
Usability

Usability is a measurement on how specific a user can use a product or a design effectively and efficiently. It is based on human behaviour and how they react to certain product or design.

Passwords
Password

Password is a string of letters, numbers or special characters that you should never share to anyone. It is used to confirm a identity of a user in a platform you are using.

Biometrics
Biometrics

Biometrics are a body measurement and calculation of humans used as a tool for authentication. It is specific and distinct that it can only work with the user so that other people don’t have access to it.

Security Policy
Security Policy

Security Policy is a set of rules and procedure required by the organization or a company for all individual accessing all IT and resources of the company.

Brain Computer Interfaces
Brain Computer Interfaces

Brain computer interfaces is a direct communication between users brain and a computer.
